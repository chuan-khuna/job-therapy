# เริ่มต้นใช้งาน Job Therapy

เครื่องมือนี้ออกแบบมาเพื่อการสะท้อนตัวเองอย่างตรงไปตรงมา ไม่ใช่เกม
ไม่มีคะแนนดีหรือแย่ มีแต่ข้อมูลที่ช่วยให้คุณเห็นความสัมพันธ์ของตัวเองกับงานชัดขึ้น

## แบบประเมินประเภทคนทำงาน

ทำครั้งแรกหนึ่งรอบเพื่อหา stereotype หลักของคุณ
แล้วกลับมาทำซ้ำเมื่อสถานการณ์งานเปลี่ยน เช่น ย้ายทีม ได้หัวหน้าใหม่
หรือรู้สึกว่าความรู้สึกต่องานไม่เหมือนเดิม

## บททดสอบความเครียดประจำวัน

บันทึกเป็นกิจวัตร — ตอบหนึ่งข้อตอนเช้า ที่เหลือก่อนนอน
จุดสำคัญไม่ใช่คำตอบของวันเดียว แต่คือแพตเทิร์นที่โผล่มาหลังจากบันทึกต่อเนื่องหนึ่งถึงสองสัปดาห์:

- เหตุการณ์เครียดเกิดซ้ำที่เดิมหรือกับคนเดิมหรือไม่
- สิ่งที่กังวลตอนเช้า ตรงกับสิ่งที่เกิดจริงบ่อยแค่ไหน
- ความรู้สึกตอนเย็นแย่กว่าหรือดีกว่าที่คาดไว้ตอนเช้า

ดูประวัติย้อนหลังได้จากปุ่ม "ประวัติ" ในแต่ละแบบประเมิน
